# AI Schedule Assistant - 任务规划

## 项目目标

开发一个 macOS 菜单栏应用，通过截图或文本选择，让 AI 自动识别并添加事件到系统原生的日历和待办清单中。

## 核心功能

1. **菜单栏应用**：顶部工具栏小图标，点击显示操作菜单
2. **内容获取**：
   - 调用系统截图工具（监听剪贴板）
   - 使用 Accessibility API 直接读取选中文本
3. **AI 处理**：使用 OpenAI 兼容 API 识别内容并生成事件描述
4. **智能分类**：AI 读取现有分类，智能匹配或创建新分类（遵循大类原则）
5. **系统集成**：添加到 macOS 原生日历和待办清单
6. **可配置模式**：用户可选择自动添加或预览确认

## 技术决策

| 决策点 | 选择 | 原因 |
|--------|------|------|
| 截图方式 | 调用系统截图工具 | 简单可靠，用户体验熟悉 |
| 文本获取 | Accessibility API | 更好的用户体验，无需手动复制 |
| AI 处理流程 | 可配置模式 | 灵活性高，满足不同用户需求 |
| 分类管理 | 完全智能分类 | 最佳用户体验，自动化程度高 |
| AI 供应商 | OpenAI 兼容 API | 最大兼容性，支持所有兼容服务 |
