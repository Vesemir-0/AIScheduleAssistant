## 设计规范总结

### 色彩使用

| 颜色 | HEX | 用途 |
|------|-----|------|
| 燕麦暖米色 | #FDF6ED | 背景色 |
| 珊瑚橘 | #EF8A6B | 主色调、主要按钮、强调 |
| 温泉水蓝 | #78BFC1 | 次要色、输入框、冰块隐喻 |
| 草木绿 | #A3C57B | 成功状态、已选中、荷叶隐喻 |
| 咖啡棕 | #5C4033 | 主要文字 |
| 咖啡棕（浅） | #8B6F5E | 次要文字 |
| 咖啡棕（更浅） | #A89080 | 三级文字 |

### 圆角规范

- 超大圆角：24pt（主要卡片、按钮）
- 大圆角：18pt（次要卡片）
- 中圆角：12pt（小组件、徽章）
- 小圆角：8pt（输入框内部元素）

### 间距规范

- Tiny: 4pt
- Small: 8pt
- Medium: 12pt
- Regular: 16pt
- Large: 20pt
- Extra Large: 24pt
- Huge: 32pt

### 阴影规范

- 暖色阴影：珊瑚橘 15% 透明度，半径 12pt，Y 偏移 4pt
- 冷色阴影：水蓝色 12% 透明度，半径 12pt，Y 偏移 4pt
- 中性阴影：咖啡棕 8% 透明度，半径 8pt，Y 偏移 2pt

## 动画效果

### 已实现的动画

1. **按钮点击**：缩放到 0.95 + 弹性动画（response: 0.3, damping: 0.7）
2. **卡片选中**：边框从无到有 + 阴影增强（0.3s ease-out）
3. **输入框聚焦**：边框颜色加深（0.2s ease-in-out）
4. **Hover 状态**：背景色淡入（0.2s ease-in-out）

### 建议添加的动画（未来优化）

3. **成功状态**：草木绿对勾弹性动画
4. **页面切换**：淡入淡出 + Y 轴位移

## 设计原则遵循情况

✅ **无攻击性** - 所有元素使用超大圆角，柔和色彩
