## 🎨 设计系统核心

### 色彩

| 颜色 | HEX | 用途 |
|------|-----|------|
| 燕麦暖米色 | #FDF6ED | 背景色 |
| 珊瑚橘 | #EF8A6B | 主色调 |
| 温泉水蓝 | #78BFC1 | 次要色 |
| 草木绿 | #A3C57B | 成功状态 |
| 咖啡棕 | #5C4033 | 主要文字 |

### 圆角

- 超大圆角：24pt
- 大圆角：18pt
- 中圆角：12pt
- 小圆角：8pt

### 间距

- Tiny: 4pt
- Small: 8pt
- Medium: 12pt
- Regular: 16pt
- Large: 20pt
- Extra Large: 24pt
- Huge: 32pt
