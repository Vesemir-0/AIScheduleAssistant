# AI Schedule Assistant UI 重设计方案
## 温泉水豚主题 - 松弛感设计系统

---

## 一、设计理念

**核心概念**：从"信息冰块"到"日程荷叶"的温暖转化过程

用户体验就像水豚泡温泉——将混乱的信息（冰块）投入温暖的AI池塘，慢慢融化成清晰的日程卡片（荷叶），整个过程充满治愈感和松弛感。

## 三、形状与材质 (Shapes & Materials)

### 圆角规范
- **超大圆角**: 24pt（主要卡片、按钮）
- **大圆角**: 18pt（次要卡片）
- **中圆角**: 12pt（小组件、徽章）
- **小圆角**: 8pt（输入框内部元素）

### 材质效果
1. **磨砂玻璃（冰块隐喻）**
   - 用于：输入框、加载状态弹窗
   - 效果：半透明背景 + 轻微模糊 + 水蓝色边框

2. **软投影（荷叶漂浮）**
   - 用于：所有卡片
   - 效果：Y轴偏移4-8pt + 暖色/冷色弥散阴影

3. **渐变光泽**
   - 用于：主要按钮
   - 效果：珊瑚橘渐变 + 轻微高光

## 五、各页面设计方案

### 1. MenuBarView（菜单栏弹出菜单）

**当前问题**：
- 纯白背景，缺乏温度
- 图标和文字间距紧凑
- 无视觉层次

**重设计方案**：
```
┌─────────────────────────────┐
│  [水豚图标]  AI 日程助手      │  ← 顶部品牌区（暖米色背景）
├─────────────────────────────┤
│                             │
│  🧊 截图获取                 │  ← 冰块图标（磨砂玻璃效果）
│     轻松捕捉屏幕信息          │
│                             │
│  ✏️ 文本输入                 │  ← 铅笔图标
│     粘贴或输入日程文本        │
│                             │
├─────────────────────────────┤
│  ⚙️ 设置                     │
│  ℹ️ 关于                     │
├─────────────────────────────┤
│  👋 退出                     │
└─────────────────────────────┘
```

**视觉特点**：
- 背景：暖米色渐变
- 按钮：超大圆角（24pt）+ hover时珊瑚橘高亮
- 图标：使用SF Symbols + 自定义颜色
- 分隔线：咖啡棕10%透明度

### 2. TextInputWindow（文本输入窗口）

**当前问题**：
- 输入框边框生硬
- 缺乏"冰块融化"的视觉隐喻

**重设计方案**：
```
┌─────────────────────────────────────┐
│                                     │
│      🧊 投入你的信息冰块              │  ← 标题（咖啡棕）
│      粘贴或输入包含日程的文本          │  ← 副标题（次要文字色）
│                                     │
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │  [磨砂玻璃质感输入框]         │   │  ← 冰块隐喻
│  │  带有水蓝色边框               │   │     半透明背景
│  │  + 轻微模糊效果               │   │     圆角24pt
│  │                             │   │
│  └─────────────────────────────┘   │
│                                     │
│     [取消]          [🔥 开始融化]    │  ← 按钮（珊瑚橘）
│                                     │
└─────────────────────────────────────┘
```

**交互动效**：
- 点击"开始融化"后，输入框边框从水蓝色渐变到珊瑚橘
- 水波纹扩散动画（0.6s ease-out）

## 十、视觉参考

### 色彩情绪板
```
🧡 珊瑚橘 #EF8A6B  ━━━━━  温暖、活力、蒸汽
💙 温泉水蓝 #78BFC1 ━━━━━  清凉、透明、冰块
💚 草木绿 #A3C57B  ━━━━━  治愈、完成、荷叶
🤎 咖啡棕 #5C4033  ━━━━━  温柔、陪伴、水豚
🤍 燕麦米 #FDF6ED  ━━━━━  松弛、纸张、温泉
```

### 关键词
- 松弛感 (Relaxed)
- 治愈系 (Healing)
- 温暖 (Warm)
- 圆润 (Rounded)
- 漂浮 (Floating)
- 融化 (Melting)
- 呼吸感 (Breathing Space)

---

**设计原则总结**：
1. **无攻击性** - 超大圆角，柔和色彩
2. **有温度** - 暖色主导，避免冷硬
3. **有层次** - 软投影，卡片漂浮
4. **有呼吸** - 充足留白，宽松间距
5. **有隐喻** - 冰块→温泉→荷叶的视觉叙事
